It dynamically showcases my 5+ years of experience as a **Senior Data Engineer**, architecting scalable cloud-native data platforms, building enterprise ETL/ELT pipelines, and modernizing legacy infrastructures natively on the Azure ecosystem.

The portfolio is designed with a premium, futuristic dark-mode aesthetic featuring interactive 3D elements powered by React Three Fiber and smooth GSAP scroll animations.
